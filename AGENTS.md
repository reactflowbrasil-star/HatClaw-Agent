# Project instructions

- This workspace is permanently associated with the GitHub repository `reactflowbrasil-star/HatClaw-Agent`.
- Use `main` as the default branch unless the user requests another branch.
- After every completed code or project-file change, verify the relevant work, stage only the files belonging to that change, create a clear Git commit, and push the current branch to `origin`.
- Never include unrelated user changes, generated secrets, credentials, or local environment files in a commit.
- If commit or push is blocked, report the exact blocker and keep the local changes intact.
